Langkah Instalasi
1
Install VirtualBox di PC Server Sekolah
Download VirtualBox versi 7.x dari USB installer yang disediakan vendor. Jalankan installer dan klik Next → Next → Install → Finish. Tidak perlu mengubah opsi apapun.

Minimum PC Server  RAM 4 GB+  HDD 100 GB+  Windows 10/11 (64-bit)
2
Buat Virtual Machine & Attach File VHD
Buka VirtualBox → klik New → isi Name: CBT-Server → Type: Linux → Version: Debian (64-bit) → Next.
RAM: minimal 2048 MB, Processor: 2 CPU → Next.
Pilih "Use an Existing Virtual Hard Disk File" → klik ikon folder → cari file cbt-server.vdi dari USB → Finish.
3
⚙️ Setting Network Adapter (PALING PENTING)
Klik kanan VM → Settings → Network, lalu atur seperti berikut:
🔴 Adapter 1 — LAN Sekolah
Enable✔ Centang
Attached toBridged Adapter
Name→ Pilih NIC ke Switch
Pilih NIC yang statusnya "No Network Access" atau yang terhubung ke switch/hub sekolah. Lihat di Control Panel → Network Connections → cari Ethernet dengan kabel ke switch.
🟢 Adapter 2 — Internet
Enable✔ Centang
Attached toNAT
Name— (kosong)
NAT = VHD dapat akses internet melalui koneksi host. Digunakan untuk update otomatis CBT.
💡 Cara cek NIC yang benar: Control Panel → Network and Internet → Network Connections → tampilkan sebagai Details → cari kolom Connectivity = "No network access" dengan Device Name = nama NIC ke switch sekolah.
4
✅ Nyalakan VM — Robot Berjalan Otomatis
Klik Start pada VM. Layar konsol VirtualBox akan menampilkan animasi dan kemudian menunjukkan IP address server yang sudah dikunci otomatis.

Tidak perlu login, tidak perlu mengetik apapun. Robot internal CBT akan:
  ① Mendeteksi IP dari router sekolah (DHCP)
  ② Mengunci IP tersebut sebagai IP statis permanen
  ③ Menampilkan URL akses untuk siswa
🚀
Server CBT Siap Digunakan
Catat IP yang tampil di layar konsol. Informasikan ke siswa untuk membuka browser dan mengetik URL di bawah ini.
IP akan selalu sama setiap VHD dinyalakan — tidak perlu diatur ulang.
